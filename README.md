# Vacations Website
A web application for seamless vacation planning, offering easy booking and navigation. 

**Backend Technologies**
- [Drogon Web Framework (C++)](https://drogon.org/)
- JWT
- PostgreSQL

Frontend Technologies
- Next.js
- [shadcn/ui](https://ui.shadcn.com/)
